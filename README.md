# BriefFlow

BriefFlow is a mobile-first multi-page HTML prototype for design students managing non-linear creative briefs.

## Pages

- `login.html`
- `calendar-brief-tracking.html`
- `creative-canvas.html`
- `reflection-feedback-ai.html`
- `process-output.html`

## Local Run

Open `login.html` directly in a browser, or open `index.html` to redirect to the login page.
